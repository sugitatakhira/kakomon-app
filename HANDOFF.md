# 過去問活用Webアプリ 引き継ぎ書

## このプロジェクトは何か
臨床検査技師 国家試験の過去問を登録・検索・テスト編成・印刷できる、**1ファイル完結のHTML Webアプリ**。
ブラウザで開くだけで動き、ネット接続も不要（CSV取り込み時の外部ライブラリを除く）。データはブラウザのlocalStorageに保存される。

教育ソフトウェア社の「過去問活用ソフト」を参考に、個人で同等のものを自作している。

## データモデル（問題1件）
```js
{
  id: "k72午前A001",        // 一意ID
  text: "[午前A1] 設問文…",  // 図問題は末尾に【別冊No.x 図】
  choices: ["選択肢1", ...], // 2〜5個
  answers: [2, 3],           // 0始まりの正解インデックス配列（複数正解対応）。採点除外は []
  explanation: "",           // 解説（採点除外問題は注記が入る）
  field: "臨床生理学",        // 科目（正式名称）
  year: "第72回(2026)",
  difficulty: "普通",        // 易/普通/難
  image: "data:image/jpeg;base64,..." // 任意。別冊図など
}
```
CSVの列は: 設問 / 選択肢1〜5 / 正解番号(例「3,4」) / 解説 / 分野 / 年度 / 難易度。
CSVの「正解番号」は**1始まり**、アプリ内部の answers は**0始まり**なので変換に注意。

## 重要な決定事項・ハマりどころ
1. **科目タグは問題番号で固定**（午前・午後 共通のブロック構造。出題基準の配点と一致＝合計200）:
   - 1〜10: 臨床検査総論
   - 11〜15: 臨床検査医学総論
   - 16〜28: 臨床生理学
   - 29〜44: 臨床化学
   - 45〜58: 病理組織細胞学
   - 59〜67: 臨床血液学
   - 68〜78: 臨床微生物学
   - 79〜89: 臨床免疫学
   - 90〜94: 公衆衛生学
   - 95〜100: 医用工学概論
2. **複数正解**（「2つ選べ」問題）に対応済み。午後B28は採点除外問題で answers=[]。
3. **別冊画像の抽出方法**: 厚労省PDFの別冊（`_02`ファイル）は本文が画像。`pdftoppm -png -r 150` で各ページをラスタ化 → 上下左右マージン(トリムマーク/フッター)をcrop → 長辺900pxに縮小 → JPEG quality≈0.68 → base64でdata URL化。
   別冊は **1ページ＝1つのNo.** で、**ページ番号 = 別冊No. + 4**（先頭4ページが表紙等）。各ページ上部に「No.◯（問題◯）」と書かれている。
4. **ダイアログ注意**: `confirm()`/`alert()` はプレビュー/サンドボックスでブロックされ削除等が止まる。**自前の確認モーダル＋トースト**に置き換え済み。新機能でもこれらは使わないこと。
5. **localStorage制約**: 画像を base64 で大量に持つと容量上限(数MB)に当たり得る。第72回で約2.2MB。年度を増やすなら IndexedDB への移行を検討。
6. CSV取り込みは PapaParse をCDNから読み込む（ここだけ要ネット）。完全オフライン化するならライブラリも埋め込む。

## 第71回のビルド（PDFからの自動化手順・重要）
72回は手入力だったが、71回は厚労省PDFから半自動で構築した。`build71/` に素材を置いた。
**ハマりどころと対策（68〜70回でも同じはず）:**
1. **問題本文PDF（a_01/b_01）はテキスト抽出に `poppler-data` が必須**。入れる前は「Missing language pack for 'Adobe-Japan1'」で日本語が出ない。
   `apt-get install -y poppler-data` 後に `pdftotext -layout` で日本語本文が取れる。
2. **斜体ラテン語（菌名・遺伝子名 *EGFR*/*BRCA1* 等）はフォントに ToUnicode が無く脱落する**（uni=no の RyuminPr6-Regular / B2Math）。
   pdftext では空白or無音で消えるので、**該当箇所だけ画像で補完**する（`build71/overrides.json` に手修正を集約）。
   脱落箇所の検出は ①空選択肢 ②`組合せ`問題 ③ギャップ検出 ④tesseract(eng)でラテン語をOCRしpdftextと差分、で洗い出す（設問内に埋もれた菌名はこれで発見）。
3. **本文ページの画像化は Ghostscript で**。poppler(pdftoppm)は本文フォントを描画できず白紙になる。
   `gs -sDEVICE=png16m -r150 -sOutputFile=p-%02d.png a_01.pdf` ならOK。
4. **別冊（a_02/b_02）は No.=ページ番号−4**（先頭4ページ表紙）。A/B分割は同一ページに両方載る。波形系（心電図/脳波）は**90°回転**して印刷されているので `ROTATE_270` で正立させる。
   抽出はトンボ/ヘッダ(No.X)/フッタを除いてautotrim→長辺900px→JPEG q72→base64（`build71/extract_beppu.py`）。
5. **設問内インライン図/表**（回路図・論理図・クロスミキシング表・補体表）は別冊ではない。
   図（A95回路・B98論理）は問題ページから切り出して画像添付、表（B61・B82）は本文に文章化して埋めた。
6. 科目は72回と同じ**問題番号の固定ブロック**で割当（個別fieldは未使用）。年度は`第71回(2025)`。採点除外なし。

**再ビルド:** `build71/` で `python3 build71.py`（要 `/tmp/k71/ans71.json`・別冊画像）→ `python3 gen_html.py`。
素材: `am_parsed.json`/`pm_parsed.json`(本文自動抽出)、`overrides.json`(手修正)、`ans71.json`(正答)、`parse.py`/`extract_beppu.py`(抽出)。
